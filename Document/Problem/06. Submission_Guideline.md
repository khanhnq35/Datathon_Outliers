# 4. Hướng dẫn Nộp bài

## Checklist nộp bài
Mỗi đội cần hoàn thành và nộp đầy đủ các mục sau:

1. **Nộp kết quả dự báo trên Kaggle**
   - Link: [Kaggle Competition](https://www.kaggle.com/competitions/datathon-2026-round-1)

2. **Báo cáo (Report)**
   - Định dạng: Sử dụng template LaTeX của NeurIPS.
   - Giới hạn: Tối đa **4 trang** (không tính references và appendix).
   - Nội dung:
     - Phần 2: Trực quan hoá và phân tích dữ liệu.
     - Phần 3: Phương pháp tiếp cận, pipeline và kết quả thực nghiệm.
   - **Đính kèm link GitHub repository** trong báo cáo.

3. **Form nộp bài chính thức**
   - Điền đáp án trắc nghiệm (Phần 1).
   - Tải lên file báo cáo (PDF).
   - Cung cấp link GitHub và link Kaggle submission.
   - Ảnh chụp thẻ sinh viên của tất cả thành viên.
